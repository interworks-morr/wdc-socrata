How to use:

  1. Find a Socrata dataset from the Open Data Network or by other means.
  2. Enter the Socrata site (e.g. soda.demo.socrata.com or data.nasa.gov) and the dataset ID (e.g. 4tka-6guv or gh4g-9sfh) into the appropriate fields.
  3. If the dataset needs to be filtered, enter the conditions into the where clause field, otherwise leave blank. See the documentation for syntax.
  4. If the dataset needs to be limited, enter the number of records in the limit field, otherwise leave blank.
